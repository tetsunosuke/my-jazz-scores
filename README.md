# 横浜市歌 - Jazz Arrange Player

このプロジェクトは、横浜市民に広く親しまれている「横浜市歌」をモダンなジャズ風にアレンジし、ブラウザ上で楽譜の表示およびインタラクティブな音源再生（試聴）を行えるWebアプリケーションです。

## 特徴
1. **ジャズ・リハーモナイズ**:
   - キーを `Eb`（変ホ長調）に設定。
   - 循環コード進行である `Ebmaj7 - C7(b9) - Fm7 - Bb7`（1-6-2-5）を軸に、セカンダリードミナント（`Eb7 -> Abmaj7`）やディミニッシュコード（`Adim7 -> Eb/Bb`）などを織り交ぜた洗練されたコード進行を適用。
2. **楽譜レンダリングと演奏同期**:
   - `abc.js` ライブラリを使用してブラウザ上に楽譜を美しく表示。
   - 再生中は、現在演奏されている音符が楽譜上でリアルタイムにハイライト（追従）されます。
3. **MIDI/Synth 再生機能**:
   - アコースティックピアノ、アコースティックベース、アルト/テナーサックスの音色を合成してスウィングジャズ伴奏をシミュレート。
   - メロディラインに加え、コード進行に沿ったバッキングとウォーキングベースラインの伴奏を自動生成します。
4. **インタラクティブなコントロール**:
   - テンポ（BPM）をスライダーで自由に変更可能。
   - 音量の調整や、アレンジ版とオリジナル（原曲）の楽譜のワンクリック切り替えが可能です。
5. **プレミアムなデザイン**:
   - 深みのあるダークカラーと半透明なガラスモルフィズム風のカードを用いたモダンでプレミアムなUI。

## 実行方法

本アプリはHTML/CSS/JavaScriptのシングルページアプリケーション（SPA）です。外部ライブラリはすべてCDNからロードされるため、ローカルで簡易サーバーを起動してブラウザで開くだけで動作します。

### 方法1: `npx serve`（推奨）
Node.jsがインストールされている場合、プロジェクトのルートディレクトリで以下のコマンドを実行します。
```bash
npx serve
```
実行後、ターミナルに表示されるURL（例: `http://localhost:3000` など）にブラウザでアクセスします。

### 方法2: Pythonの簡易サーバー
Pythonがインストールされている場合、以下のコマンドで起動できます。
```bash
# Python 3
python -m http.server 8000
```
その後、ブラウザで `http://localhost:8000` にアクセスします。

## コードアレンジ詳細
- **Aセクション（前半）**:
  メロディの素朴さを残しつつ、各小節にジャズの代理コードを配置。
  `| Ebmaj7 | C7(b9) | Fm7 | Bb7 |`
  の進行が滑らかなベースの流れを作り出します。
- **Bセクション（後半）**:
  「されば港の数多かれど」のパートからは、ドラムやウォーキングベースがより強調されるスウィング感を意識したリズム設計。終盤の「とま屋の煙」では `Abmaj7 -> Adim7 -> Eb/Bb -> C7(b9)` の美しいバラード的なハーモニー解決を導入しています。
